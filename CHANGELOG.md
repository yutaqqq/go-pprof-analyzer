## [0.1.0] — 2026-04-22

### Добавлено

- **Парсер профилей** — загрузка heap/cpu/goroutine/mutex профилей через `github.com/google/pprof/profile`; автоопределение типа по именам sample-type
- **Heap-анализатор** — топ аллокаторов по `alloc_space`/`inuse_space`; flat и cumulative значения; вычисление процентов
- **CPU-анализатор** — горячие пути по flat/cumulative CPU-времени
- **Анализатор горутин** — группировка по стеку вызовов, сортировка по количеству
- **Детектор утечек** — сравнение двух goroutine-снимков во времени; выявление стеков с растущим счётчиком
- **Diff-вьюер** — поддекционное сравнение двух профилей: выросло / уменьшилось / новые / исчезнувшие аллокаторы
- **Движок рекомендаций** — автоматические советы: sync.Pool для частых аллокаций, easyjson для JSON, strings.Builder для fmt.Sprintf, make(n) для growslice
- **Markdown-отчёт** — таблицы с топ-аллокаторами, горячими путями, утечками, diff
- **JSON-экспорт** — полный отчёт в машиночитаемом формате
- **CLI на Cobra** — три команды: `analyze`, `diff`, `leak`; флаги `--top`, `--output`, `--format`
- **GitHub Actions CI** — `go test -race` + golangci-lint на каждый push
